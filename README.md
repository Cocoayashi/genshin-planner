### Genshin Planner

Up until recently, the live-service video game "Genshin Impact" had specified days of the week that certain level up materials were available to obtain.
The game made it difficult to figure out which materials you should work on getting that day. 

I made this app to simplify the process of figuring out which materials I needed to obtain on which days of the week. 
The idea was that I would:
1.) Type in which characters I was levelling up 
2.) Then the app would return what materials I needed corresponding to each day of the week

Luckily, Genshin Impact has since implemented a way to do this in-game without needing an external app like this :)
